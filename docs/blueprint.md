# **App Name**: AvivaQuest

## Core Features:

- Cross-Platform Access: Responsive design for access on both desktop and mobile devices, ensuring usability for all new hires.
- Single Question Display: The app delivers onboarding quiz questions one at a time, in order to maintain focus.
- Animated Progress Indicator: Provides an animated progress bar to show advancement in the onboarding quizzes.
- Adaptive Quiz Selection: Uses branching logic to direct each new hire through one of two quiz tracks (BA and ATN) based on role.
- Narrative-Driven Missions: Uses distinct opening narratives and mission theming based on the selected quiz (BA or ATN) in order to set context.
- AI Motivational Feedback: A motivational phrase will be chosen by an AI tool, and shown on the result screen. This phrase will depend on the quiz topic and final score.

## Style Guidelines:

- Primary color: Medium-dark cyan (#46A5B3), a professional, calming color slightly suggestive of money or growth.
- Background color: Very light cyan (#E0F4F7)
- Accent color: Green (#63B346), a natural choice to complement the cyan.
- Body and headline font: 'Inter', sans-serif, for a clean and modern feel, readable in extended passages.
- Incorporate Aviva's existing icons where possible, supplemented by flat, friendly icons for navigation and feedback.  Keep the number of icons small.
- Simple transitions and progress bar animations to engage users without overwhelming them.
